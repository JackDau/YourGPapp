## Interaction Notes

1. **Form Validation**: Real-time validation as user types
2. **Loading States**: Button shows spinner during API calls
3. **Error Handling**: Inline errors below fields, not alerts
4. **Accessibility**: All inputs have labels, error messages linked to fields
5. **Mobile Responsive**: Single column layout scales from 320px to desktop
