## Responsive Behavior

### Mobile (< 768px)
- Single column layout
- Bottom navigation visible
- Hamburger menu for additional nav
- Quick actions in horizontal scroll or 3-column grid

### Tablet (768px - 1024px)
- Two column layout for quick actions + activity
- Bottom navigation or side navigation
- More spacious cards

### Desktop (> 1024px)
- Side navigation replaces bottom nav
- Dashboard becomes multi-column
- Quick actions + activity side by side
- Wider content area with max-width constraint
